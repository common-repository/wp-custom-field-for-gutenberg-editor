== Description ==

The plugin facilitates the user to continue using the functionality of custom fields along with Gutenberg WordPress editor.
It allows the custom field boxes to be displayed on all the posts and pages that have enabled Gutenberg functionality. 
The content of the post and pages can be easily managed on the Gutenberg edit screen at the backend. 

= Key Features: =

* Simple to use
* Retrieve and display existing fields
* Addition of multiple fields without updating or refreshing the post.
* Each custom field has an update and delete button
* Works seamlessly with other WordPress Plugins
* Exclusion of custom fields that are either protected or hidden
* Exclusion of custom fields with null values

* Download the plugin
* Upload the folder "wp-custom-field-for-gutenberg-editor" to wp-content/plugins (or upload a zip through the Wordpress admin)
* Activate and enjoy!
